Step-by-Step Setup

Getting started with Modus Web Components is straightforward:

  1. Include the CSS: 
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@trimble-oss/moduswebcomponents/modus-wc-styles.css" />
  2. Load the JavaScript: 
    <script type="module">
  import { defineCustomElements } from "https://cdn.jsdelivr.net/npm/@trimble-oss/moduswebcomponents/loader/index.js";
  defineCustomElements();
</script>
  3. Set the theme: 
    <html data-theme="modus-modern-light">
  4. Start using components!
